## 基础环境

- 开发工具：VSCode
- git可视化工具: SourceTree
- node: ≥v16.6.1
- npm: ≥7.20.6
- 开发语言：JavaScript, TypeScript

### node/npm 安装

1. 安装 node/npm

```
brew install node
```

2. node 升级和切换版本

使用 n 管理 node 版本

```
npm install -g n

n 16.6.1
```

### taro 安装

```
npm install -g @tarojs/cli
```

### React Native

1. App Store 下载 xcode

2. cocoapods 环境

```
# 安装
brew install cocoapods

# 检查安装是否成功，查看 cocoapods 版本
pod --version
```

> cocoapods 版本 >=1.10.2
